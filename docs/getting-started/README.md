# 快速入門 — 選擇你的角色

Dynamic Alerting 平台有三種主要使用角色。選擇最符合你的角色，開始閱讀對應的入門指南：

| 你是誰？ | 你想做什麼？ | 入門指南 |
|---------|------------|---------|
| **Platform Engineer** | 部署叢集、設定 Prometheus/Alertmanager、管理 Rule Pack | [Platform Engineer 入門](for-platform-engineers.md) |
| **Domain Expert** | 定義閾值、設計告警規則、管理路由策略 | [Domain Expert 入門](for-domain-experts.md) |
| **Tenant User** | 查看告警、調整自己租戶的閾值、理解告警含義 | [Tenant User 入門](for-tenants.md) |

> 不確定？從 [Tenant User 入門](for-tenants.md) 開始 — 它涵蓋最基本的概念。

**互動式嚮導：** 也可以使用 [Setup Wizard](wizard.jsx) 透過問答方式引導你完成初始設定。
