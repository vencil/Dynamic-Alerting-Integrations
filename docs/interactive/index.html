Dynamic Alerting — Interactive Tools

Browser-based tools for the Multi-Tenant Dynamic Alerting platform. No installation required — runs entirely in your browser.

Tenant YAML Validator
Paste a tenant YAML configuration and get instant validation feedback. Includes pre-built templates for MariaDB, PostgreSQL, Redis, and Kafka.
validation yaml live preview
CLI Playground
Build da-tools CLI commands interactively. Select a command, configure flags and arguments, then copy the ready-to-run command.
cli da-tools docker
Rule Pack Selector
Select the Prometheus rule packs you need, see rule counts, and generate the YAML configuration snippet for your monitoring stack.
prometheus rule packs config gen
Getting Started Wizard
Answer a few questions about your role and goals to get a personalized reading path through the documentation.
onboarding docs guided

Part of the Dynamic Alerting project (v2.0.0-preview). Components load via client-side JSX — no server required.
